Basic stuff

bold text :

use asterics like this: **its bold**

italic text :

use underscore like this: _its italic_

bold & italic :

use asterics and underscore **_bold&italic_**

header: use hash mark (the more the smaller the header become)

#header
##header
###header
####header

Links : Inline links

[Visit GitHub!](www.github.com)

Links : Reference links

 Here's [a link to something else][another place].
 Here's [yet another link][another-link].
 And now back to [the first link][another place].

 [another place]: www.github.com
 [another-link]: www.google.com

Blockquotes : Use this sign in front of Text: First you need to change *.css file and add:

blockquote {
  background: #f9f9f9;
  border-left: 5px solid #ccc;
  margin: 1.5em 10px;
  padding: 0.5em 1.5em;
}

than you can make blockquotes with this syntax:

<blockquote>
some bockquotes
</blockquote>
some bockquotes
